# Features

Flashwars is a comprehensive study and game platform with the following capabilities.

## Study Sets

- Terms with optional images and audio pronunciations
- Folders one level deep with tagging and color coding
- Privacy modes: private, link-only, public
- Import/export support for CSV, Quizlet, and Anki formats
- Collaborative editing with version history
- Duplicate detection and merge suggestions

## Study Modes

### Flashcards

- Smart spaced repetition, alphabetical, random, and original position ordering strategies
- Skip specific terms functionality
- Returns front, back, and stable identifier for progress tracking
- Customizable flip animation speed and auto-advance options
- Keyboard shortcuts for navigation and marking difficulty

### Learn

- Adaptive rounds based on prior performance
- Configurable item count, item types, and matching pair count
- Prioritizes due and unseen items from scheduler
- Prevents term reuse within a single round
- Supports exclusion lists
- Hint system with progressive reveals
- Mastery tracking with visual progress indicators

### Test

- Fixed-length assessments generated in one call
- Same configuration options as Learn mode
- Prioritizes due and unseen items for authenticated users
- Full set consideration for anonymous users
- Entire test precomputed upfront
- Timed and untimed modes
- Instant feedback or end-of-test review options
- Retake with missed items only

## Question Types

### Multiple Choice

- Exactly four options with one correct answer and three plausible distractors
- Deterministic shuffling via optional seed
- Clear, neutral stems with mutually exclusive distractors
- Image-based questions support

### True/False

- Atomic, unambiguous statements
- Balanced distribution across rounds
- No trick wording
- Explanation text for incorrect answers

### Free Text

- Canonical answers with optional aliases
- Case-insensitive and punctuation-trimmed normalization
- Post-grading feedback with rationale
- Partial credit for close matches
- Voice input support

### Matching

- Fixed pair count configuration
- Independent column randomization
- Unambiguous term-to-definition pairs
- No duplicates within items
- Drag-and-drop or click-to-select interfaces

## Progress and Analytics

- Track hardest terms and per-mode trends
- Spaced repetition scheduler for due items
- CSV export of progress data
- Learning streaks and achievement badges
- Time spent studying metrics
- Accuracy heatmaps by day/week/month
- Personalized study recommendations

## Classroom

- Classes, sections, and enrollments
- Assignments with due dates and grace periods
- Gradebook and activity monitoring
- Announcement system with read receipts
- Group study sessions with live collaboration
- Peer review capabilities
- Parent/guardian view access

## Organizations

- Organizations with roles: Owner, Admin, Member
- Domain allowlists and organization analytics
- Centralized billing and license management
- Custom branding options
- SSO/SAML integration support

## Flash Wars Games

- Duel, Party, and Class host modes
- Server authoritative timing and scoring
- Up to 40 players per game
- Tournament brackets and seasonal competitions
- Powerups and special abilities
- Real-time chat with moderation
- Spectator mode

## Leaderboards

- Leaderboards per set and per game
- Anti-spam safeguards and tie-break by earliest time
- Weekly, monthly, and all-time views
- Friend-only leaderboards
- School and class rankings

## Admin Console

- User search and moderation of public sets
- Organization management
- Feature flags and health monitoring
- Content flagging and review queue
- Usage analytics dashboard
- System performance metrics
- Bulk user operations

## Notifications

- In-app and email notifications for invites, due items, summaries, and results
- Push notifications for mobile apps
- Customizable notification preferences by type
- Digest options (daily, weekly)
- Do not disturb scheduling

## AI Features

- Distractor generation for multiple choice questions
- Semantic grading with confidence levels
- Multilingual support and on-demand translation
- Insight generation
- Auto-generated study guides and summaries
- Pronunciation assessment for language learning
- Image recognition for diagram labeling
- Personalized difficulty adjustment

## Accounts and Authentication

- User accounts with authentication and role-based access
- Social login (Google, Apple, Microsoft)
- Two-factor authentication
- Password recovery and account merging
- Privacy settings and data export compliance (GDPR)

## Accessibility

- Screen reader compatibility
- High contrast and dark mode themes
- Keyboard-only navigation
- Adjustable font sizes and spacing
- Alternative text for all images
